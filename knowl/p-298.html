First, some preliminary investigation to find a suitable \(\delta\text{.}\) Consider \(\abs{f(x)-0}\lt\varepsilon\text{:}\)

\begin{align*} \abs{\sin(x)-0}\amp\lt\varepsilon\\ \abs{\sin(x)}\amp\lt\varepsilon \end{align*}

Using the hint that \(\abs{\sin(x)} \leq \abs{x}\text{,}\) then if \(\abs{x}\lt0\text{,}\) it is automatic that \(\abs{\sin(x)}\lt0\text{.}\) This suggests we take \(\delta=\varepsilon\text{.}\)
