Solution 1.3.2.1.

  1. Using the Sum/Difference rule, we know that

    \begin{align*} \lim_{x\to 2}(f(x) + g(x)) \amp = \lim_{x\to 2}f(x) + \lim_{x\to 2}g(x)\\ \amp = 2+3 =5\text{.} \end{align*}

  2. Using the Scalar Multiple, Sum/Difference, and Power rules, we find that

    \begin{align*} \lim_{x\to 2}(5f(x) + g(x)^2) \amp = \lim_{x\to 2}(5f(x))+\lim_{x\to 2}(g(x)^2)\\ \amp = 5\lim_{x\to 2}f(x) + \mathopen{}\left(\lim_{x\to 2}g(x)\right)\mathclose{}\\ \amp = 5\cdot 2 + 3^2 = 19\text{.} \end{align*}

  3. Here we combine the Power, Scalar Multiple, Sum/Difference and Constant Rules. We show quite a few steps, but in general these can be omitted:

    \begin{align*} \lim_{x\to 2} p(x) \amp = \lim_{x\to 2}\left(3x^2-5x+7\right)\\ \amp = \lim_{x\to 2}\mathopen{}\left(3x^2\right)\mathclose{} -\lim_{x\to 2}(5x)+\lim_{x\to 2}7\\ \amp = 3\bigl(\lim_{x\to 2}x\bigr)^2-5\lim_{x\to 2}(x) +7\\ \amp = 3\cdot 2^2 - 5\cdot 2+7\\ \amp = 9 \end{align*}
