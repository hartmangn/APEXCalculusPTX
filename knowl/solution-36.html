Solution 1.

Let \(\varepsilon \gt 0\) be given. We wish to find \(\delta \gt 0\) such that when \(\abs{x-5}\lt\delta\text{,}\) \(\abs{f(x)-(-2)}\lt\varepsilon\text{.}\)

First, some preliminary investigation to find a suitable \(\delta\text{.}\) Consider \(\abs{f(x)-(-2)}\lt\varepsilon\text{:}\)

\begin{align*} \abs{f(x)+2}\amp\lt\varepsilon\\ \abs{(3-x)+2}\amp\lt\varepsilon\\ \abs{5-x}\amp\lt\varepsilon\\ \abs{x-5}\amp\lt\varepsilon \end{align*}

Since we want to start with \(\abs{x-5}\lt\delta\text{,}\) this suggests we let \(\delta=\varepsilon\text{.}\)

Now we can apply the definition.

\begin{align*} \abs{x-5}\amp\lt\delta\\ \abs{x-5}\amp\lt\varepsilon\\ -\varepsilon\lt x-5\amp\lt\varepsilon\\ -\varepsilon\lt (x-3)-2\amp\lt\varepsilon\\ \varepsilon\gt(-x+3)-(-2)\gt\amp-\varepsilon\\ \abs{(3-x)-(-2)}\amp\lt\varepsilon\text{.} \end{align*}

In other words, \(\abs{x-5}\lt\delta\) implies \(\abs{(3-x)-(-2)}\lt\varepsilon\text{.}\) This is what we needed to prove.
