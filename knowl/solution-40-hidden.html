Let \(\varepsilon \gt 0\) be given. We wish to find \(\delta \gt 0\) such that when \(\abs{x-2}\lt\delta\text{,}\) \(\abs{f(x)-7}\lt\varepsilon\text{.}\)

First, some preliminary investigation to find a suitable \(\delta\text{.}\) Consider \(\abs{f(x)-7}\lt\varepsilon\text{:}\)

\begin{align*} \abs{\left(x^3-1\right)-7}\amp\lt\varepsilon\\ \abs{x^3-8}\amp\lt\varepsilon\\ \abs{x-2}\cdot\abs{x^2+2x+4}\amp\lt\varepsilon\\ \abs{x-2}\amp\lt\frac{\varepsilon}{\abs{x^2+2x+4}} \end{align*}

Since \(x\) is near \(2\text{,}\) we can safely assume that, for instance, \(1\lt x\lt 3\text{.}\) Thus

\begin{align*} 1^2+2(1)+4\amp\lt x^2+2x+4\lt3^2+2(3)+4\\ 7\amp\lt x^2+2x+4\lt 19\\ \frac{1}{19}\amp\lt\frac{1}{x^2+2x+4}\lt\frac{1}{7}\\ \frac{\varepsilon}{19}\amp\lt\frac{\varepsilon}{x^2+2x+4}\lt\frac{\varepsilon}{7} \end{align*}

Since we need to begin the actual proof with \(\abs{x-2}\lt\delta\text{,}\) this suggests that we take \(\delta=\frac{\varepsilon}{19}\text{.}\)

Now we can apply the definition.

\begin{align*} \abs{x-2}\amp\lt\delta\\ \abs{x-2}\amp\lt\frac{\varepsilon}{19}\\ \abs{x-2}\amp\lt\frac{\varepsilon}{\abs{x^2+2x+4}}\\ \abs{x-2}\cdot\abs{x^2+2x+4}\amp\lt\varepsilon\\ \abs{x^3-8}\amp\lt\varepsilon\\ \abs{\left(x^3-1\right)-7}\amp\lt\varepsilon \end{align*}

In other words, \(\abs{x-2}\lt\delta\) implies \(\abs{\left(x^3-1\right)-7}\lt\varepsilon\text{.}\) This is what we needed to prove.
