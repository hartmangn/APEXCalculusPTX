First, some preliminary investigation to find a suitable \(\delta\text{.}\) Consider \(\abs{f(x)-(-2)}\lt\varepsilon\text{:}\)

\begin{align*} \abs{f(x)+2}\amp\lt\varepsilon\\ \abs{(3-x)+2}\amp\lt\varepsilon\\ \abs{5-x}\amp\lt\varepsilon\\ \abs{x-5}\amp\lt\varepsilon \end{align*}

Since we want to start with \(\abs{x-5}\lt\delta\text{,}\) this suggests we let \(\delta=\varepsilon\text{.}\)
